# Split Cost

A web application that determines each person's share of a group dining bill. 

Have you ever went out with a large group of people and wondered how much each group member should pay.  Split Cost could help you figure it out.  Try it out via this link: <a href="https://splitcostapp.netlify.app/" target="_blank">https://splitcostapp.netlify.app/</a>.

- CSS
